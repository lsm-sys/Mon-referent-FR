Приложение получает на входе адрес статьи на французском языке, выполняет парсинг и, по соответствующей кнопке, подключает AI и выполняет нужное действие.
Интерфейс:
Поле ввода URL франкоязычной статьи.
4 кнопки: «Posts pour X», «Posts pour Instagram», «Infographies pour Instagram», «Post pour Telegram».
Блок для отображения результата, в котором происходит генерация нужного вида ответа.
